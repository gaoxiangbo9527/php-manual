数组操作符
    示例        名称         结果
    $a + $b	    合并	        合并 $a 和 $b 。
    $a == $b	相等	        如果 $a 和 $b 具有相同的键/值对，则为 TRUE 。
    $a === $b	全等	        如果 $a 和 $b 具有相同的键/值对，且顺序相同、类型相同，则为 TRUE 。
    $a != $b	不等	        如果 $a 不等于 $b ，则为 TRUE 。
    $a <> $b	不等	        如果 $a 不等于 $b ，则为 TRUE 。
    $a !== $b	非全等	    如果 $a 不全等于 $b ，则为 TRUE 。

+ 数组操作符会将右侧数组追加到左侧数组，对于那些存在于两个数组中的键，将使用左侧数组中的值，并且右侧数组中对应的值会被忽略。
个人补充：通过上面的说明可以看出，+ 和 += 对于合并数组来说结果是一样的。
