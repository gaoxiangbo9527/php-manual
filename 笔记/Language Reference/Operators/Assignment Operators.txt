基本的赋值运算符是 "=" 。你的第一反应可能以为它是“等于”。其实不是，它实际上意味着把左侧的操作数设置为右侧的表达式的值。

赋值表达式的值就是被赋的值。也就是说，"$a = 3" 的值是 3 。

除了基本的赋值运算符外，所有的二元算术运算符、数组合并运算符和字符串运算符都有对应的“组合运算符”。这些组合运算符允许在一个表达式中使用一个值，
并且设置它的值到表达式的结果。

注意赋值运算符会将原始变量复制到新的变量中（按值赋值），因此改变一个并不会影响另一个。如果你需要在密集的循环中拷贝诸如大数组的东西，那么这也
很用。

PHP 中常见的按值赋值的一个例外就是碰到对象时，它是按引用赋值的。对象可以通过 clone 关键字显式的复制。
个人补充：其实对象也是按值赋值的，只不过复制的是对象标识符。

按引用赋值
    按引用赋值也是支持的，使用 "$var = &$othervar;" 语法即可。按引用赋值意味着两个变量最终指向相同的数据，并且没有拷贝任何东西。

    new 操作符会自动的返回一个引用，所以从 PHP 7.0.0 开始不允许按引用赋值 new 的结果，从 PHP 5.3.0 开始会导致一个 E_DEPRECATED 错误
    ，在早期版本中会导致一个 E_STRICT 错误。

    关于引用及其潜在用途的更多信息可以在手册的 引用的解释 章节中被找到。
