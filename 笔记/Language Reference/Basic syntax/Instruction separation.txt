与在 C 和 Perl 中一样，PHP 需要在每条语句的结尾用分号结束指令。
一个 PHP 代码块的结束标签会自动暗含一个分号；所以在一个 PHP 代码块的最后一行可以不用分号结束（前提是有结束标签）。
个人补充：建议每条语句后面都加上分号，即使是 PHP 代码块的最后一行。
如果代码块的结束标签后紧跟着一个换行符，那么结束标签包括这个换行符。

注意
    位于文件尾部的 PHP 代码块的结束标签是可选的，有些情况下当使用 include 或者 require 时省略掉它会更好，这样在文件的尾部就不会出现不期望
    的空白字符，从而你能在稍后往响应中添加头部信息。在使用输出缓冲时也很便利，就不会看到由被包含文件生成的不期望的空白字符。
