PHP 需要在每个语句的结尾用分号结束指令。
一个 PHP 代码块的结束标记会自动的提供一个分号；所以在一个 PHP 代码块的最后一行可以不用分号结束(前提是有写结束标记)。
如果代码块的结束标记后面紧跟着一个换行符，则结束标记包括这个换行符。

注意
    文件尾部的 PHP 代码块的结束标记是可选的，有些情况下当使用 include 或者 require 时省略掉会更好，这样在文件的尾部就不会出现不期望的空白字符，
    从而你能在稍后往响应中添加头部信息。在使用输出缓冲时也很便利，就不会看到由包含文件生成的不期望的空白字符。

C 风格的注释在碰到第一个 */ 时结束。
要确保不要嵌套 C 风格的注释。
试图注释掉一大块代码时很容易出现该错误。
