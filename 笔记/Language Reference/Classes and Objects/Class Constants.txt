在定义和使用常量的时候不需要使用 $ 符号
类常量默认的可见性是public
常量的值不能是一个变量、属性或者函数调用，必须是一个常量表达式（从PHP 5.6.0 开始才可以使用常量表达式，PHP 5.6.0 以前只能是常值）
接口中也能定义常量
从PHP 5.3.0 开始可以使用变量来引用某个类，但该变量的值不能为关键字（如 self，parent 或 static）
    $classname = "MyClass";
    echo $classname::CONSTANT . "\n"; // As of PHP 5.3.0

    $class = new MyClass();
    echo $class::CONSTANT."\n"; // As of PHP 5.3.0
类常量只针对每个类分配一次，而不是针对每个实例都分配一次

从PHP 5.3.0 开始可以使用heredoc和nowdoc语法初始化类常量
从PHP 5.5.0 开始可以使用特殊的类常量 ::class，并且能够在编译时就得到完整限定的类名，这对于具有命名空间的类尤其有用。

从PHP 7.1.0 开始才可以在类常量前加可见性修饰符，在7.1.0之前这么做会报语法错误。
