把类属性或方法声明为静态，就可以不实例化类而直接访问它们。
静态属性不能通过一个类已实例化的对象来访问（但静态方法可以）（个人补充：静态属性也能通过一个类已实例化的对象来访问）。

为了兼容 PHP 4，如果（声明静态属性和方法时）没有指定可见性，（静态）属性和方法默认为 public。

Static methods
    由于静态方法不需要创建实例化对象即可调用，所以伪变量 $this 在静态方法中不可用。

Static properties
    静态属性不可以由对象通过 -> 操作符来访问。

    在PHP 5.6.0 之前，静态属性同其他静态变量一样，只能用字面量或者常量初始化，而不能用表达式初始化。
    自PHP 5.6.0 开始，可以使用常量表达式，即只要能在编译时得到其值的表达式就可以。
