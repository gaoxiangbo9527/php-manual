服务器和执行环境信息

描述
    $_SERVER 是一个包含了诸如头信息、路径以及脚本位置等信息的数组。
    这个数组中的条目由 Web 服务器创建。不能保证每个服务器都提供所有的条目；服务器可能会忽略一些，或者提供一些没有在这里列举出来的项目。
    这些变量中的大部分都被列入了 CGI 1.1 规范中，所以应该仔细研究一下。
    注意
        在 PHP 5.4.0 之前，$HTTP_SERVER_VARS 中包含着和 $_SERVER 相同的初始信息，但它却不是一个超全局变量。
        注意，$HTTP_SERVER_VARS 和 $_SERVER 是不同的变量，PHP 处理它们的方式也不同。

更新日志
    5.4.0	移除了 register_long_arrays 指令（功能），所以 $HTTP_SERVER_VARS 不再有效。
    5.3.0	废弃了使 $HTTP_SERVER_VARS 生效的 register_long_arrays 指令。
    4.1.0	引入 $_SERVER，弃用 $HTTP_SERVER_VARS。

注意事项
    注意：$_SERVER 是一个超全局变量，或者说自动全局变量。这就意味着它可以在脚本的任意作用域中使用。不需要进行 global $variable; 就可以在
    函数和方法中访问它。
