通过使用可选的 return 语句可以返回值。任意类型的值可以被返回，包括数组和对象。return 语句会立即结束函数的执行并将控制权传回调用该函数所在的行。

注意
    如果省略 rerurn ，则会返回 NULL 值。

使用 return
    一个函数不能返回多个值，但是通过返回一个数组可以得到类似的结果。
    若要从函数中返回引用，请在函数声明中以及将返回值赋给变量时使用 & 操作符。

返回类型声明
    PHP 7 添加了对返回类型声明的支持。类似于参数类型声明，返回类型声明指定了将从一个函数返回的值的类型。返回类型声明可以使用和参数类型声明相同的类型。

    严格类型也会对返回类型声明有影响。在默认的弱模式中，如果返回的值还不是声明的返回类型，则将强制转换为正确的类型。在强模式中，返回值必须是声明的返回
    类型，否则会抛出一个 TypeError 。

    从 PHP 7.1.0 开始，通过在类型名前面前置一个问号 ? 就可以将返回值标记为可以为空。这表示函数可以返回指定的类型或 NULL。

    注意
        当覆盖父方法时，子方法必须匹配父方法上的返回类型声明。如果父方法没有定义一个返回类型，那么子方法可以定义一个。
