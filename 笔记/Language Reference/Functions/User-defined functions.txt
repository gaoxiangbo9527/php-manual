任何有效的代码都可以出现在函数中，甚至是其他的函数和类的定义。

函数名遵照和 PHP 中其他标签相同的命名规则。一个有效的函数名以字母和下划线开头，跟着任意数量的字母、数字和下划线。作为正则表达式的话，它将这样
表示：^[a-zA-Z_\x80-\xff][a-zA-Z0-9_\x80-\xff]*$

函数不需要在被引用之前被定义（个人补充，这句话的意思是函数定义的位置不一定要在函数调用位置的前面或者上面），除非函数是被条件定义的。
如果函数是被条件定义的，那么它的定义必须在调用之前被处理。

PHP 中的所有函数和类都具有全局作用域。即使在函数内被（先）定义，它们也能在函数外被调用，反之亦然。

PHP 不支持函数重载，也不能取消定义或者重新定义之前声明的函数。

注意：对于 ASCII 字符 A 到 Z，函数名是大小写不敏感的。虽然如此，但在调用函数时使用该函数声明时的形式通常来说是好的。

在 PHP 中可以递归的调用函数。

注意：超过 100-200 递归级别的递归函数/方法的调用会破坏堆栈并导致当前脚本终止。特别是，无限递归被认为是一种编程错误。
个人补充，个人做测试最大递归级别目前是 256
