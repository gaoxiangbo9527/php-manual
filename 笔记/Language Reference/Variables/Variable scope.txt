变量的作用域即定义该变量的上下文（环境）。

大部分的 PHP 变量只有一个的作用域。
这个作用域的跨度包含由 include 和 require 引入的文件。

在用户自定义的函数中，一个局部的函数作用域将被引入。在函数内部用到的任何变量，默认情况下都被限制于这个局部的函数作用域之内。
PHP 的全局变量和 C 语言有一点点不同，在 C 语言中，全局变量在函数内自动生效，除非被局部变量覆盖。这可能会引起一些问题，有些人可能不小心就改变了一个全局变量。
在 PHP 中，如果一个全局变量要在函数内使用，必须在该函数内被声明为 global 。

global 关键字
    第一种访问全局作用域中的变量的方法是使用 global 关键字。
    第二种访问全局作用域中的变量的方法是使用特殊的 PHP 预定义的 $GLOBALS 数组。
    $GLOBALS 之所以存在于任何作用域内，是因为 $GLOBALS 是一个超全局变量（这样的全局变量不用借助 global 关键字就可以引入到某个作用域内）。

    注意
        在函数外面使用 global 关键字并不是错误，如果一个文件被是在一个函数中被包含进去的，就可以这么用。

使用静态变量
    静态变量仅在存在于局部的函数作用域中，但当程序执行离开该作用域时，静态变量的值并不会丢失。

    静态变量可以用于处理递归函数。

    注意
        从 PHP 5.6 开始，你可以将表达式的结果赋值给静态变量，但是此时该表达式中不能使用任何函数，这么做会导致解析错误。
    注意
        静态（变量）声明是编译时进行解析的。
