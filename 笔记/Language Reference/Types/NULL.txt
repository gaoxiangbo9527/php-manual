NULL 值表示一个变量没有值。

一个变量在下面几种情况下会被认为是空类型：
    尚未被赋值。
    被赋值为 null 。
    被 unset() 。

语法
    空类型只有一个值，即常量 NULL ，它是不区分大小写的。
    （个人补充）建议使用小写形式，即 null 。

转换为 NULL
    使用 (unset) $var 将一个变量转换为 null 将不会删除该变量或 unset 其值，而仅是返回 null 值而已。
    PHP 7.2.0 废弃了这一特性，不建议再依赖这一特性了。
    （个人补充）空类型既然表示没有值的变量，那么其他类型的变量，即有值的变量转换为空类型确实不太合理，从这个角度理解的话，废弃这个特性是合理的。
