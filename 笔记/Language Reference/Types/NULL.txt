NULL 值表示一个变量没有值。

一个变量在下面几种情况下会被认为是空类型：
    尚未被赋值。
    被赋值为 null 。
    被 unset() 。

语法
    空类型只有一个值，即常量 NULL ，它是不区分大小写的。
    （个人补充）建议使用小写形式，即 null 。

转换为 NULL
    显式转换
        (unset)
            使用 (unset) $var 将一个变量转换为 null 不会删除该变量或 unset 其值，而仅是返回 null 值而已。
        （个人补充）而 unset() 函数会删除该变量并会改变变量的类型。
    隐式转换
        无

    PHP 7.2.0 废弃了“转换为NULL”这一特性，不建议再依赖这一特性了。
