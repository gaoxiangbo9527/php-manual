PHP 支持 10 种原始类型。
    四种标量类型：
        boolean
        integer
        float
        string
    四种复合类型：
        array
        object
        callable
        iterable
    两种特殊类型：
        resource
        NULL

为了易读性原因，本手册还引入了一些伪类型：
    mixed
    number
    callback
    array|object
    void
还有伪变量 $... 。

实际上 double 和 float 是相同的，由于历史遗留的原因，这两个名称同时存在。

变量的类型通常不是由编程者设置的，而是由 PHP 根据该变量在运行时使用的上下文决定的。

注意
    想要检查一个表达式的类型和值，可以使用 var_dump() 函数。
    想要获取一个类型的人类易读表示形式用于调试，可以使用 gettype() 函数。
    想要检查某个特定的类型，不要使用 gettype() 函数，而应当使用 is_type 函数。

想要将一个变量强制的转换为特定的类型，可以使用 强制转换 或者 settype() 函数。
