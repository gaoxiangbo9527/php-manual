实际上 callable 应该算作是一个伪类型，它真实的类型应该是对象。

从 PHP 5.4 开始可以使用 callable 类型提示来表示 callbacks 。
回调函数可以是简单函数，还可以是对象的方法和静态类的方法。

传递
    PHP 简单函数是使用其函数名对应的字符串传递的。
    可以使用内置函数或者自定义函数，不过不能使用语言结构，例如：array()、echo、empty() eval()、exit()、isset()、list()、print 或 unset() 。
    除了普通的用户自定义函数外，也可传递 匿名函数 给回调参数。

    已实例化的对象的方法是以数组传递的，数组中下标 0 包含该对象，下标 1 包含方法名。
    允许在同一个类里访问其 protected 和 private 方法。

    静态类的方法也是以数组传递的，不过不实例化该类的对象就可以传递，只是下标 0 中包含是类名而不是对象，下标 1 仍包含的是方法名。
    从 PHP 5.2.3 起，还可以这样传递：'ClassName::methodName' 。
注意
    使用 call_user_func() 和 call_user_func_array() 函数注册了多个回调时，如果在前一个回调中有未捕获的异常，其后的回调将不再被调用。
