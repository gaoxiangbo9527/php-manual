语法
    想要指定一个布尔型的字面量，使用常量 TRUE 或者 FALSE ，它们不区分大小写。
    （个人补充）通常建议使用全小写形式，即 true 和 false 。

    通常返回布尔类型值的操作符的结果会被传递给控制流程使用。

转换为布尔值
    显式转换
        (bool) 或 (boolean)
        boolval()
    隐式转换
        如果一个操作符、函数或者控制结构需要一个布尔类型的参数，该值会自动转换。

    当转换成布尔值时，以下值会转换为 false ：
        boolean     false
        integer     0（非零值都会转换成 true ）
        float       0.0（NAN 也会被转换成 true ）
        string      "" 或者 "0"（从编程语言的角度来说，"0"转换成 false 是不合理的，而从实用性角度来说 "0" 转换成 false 确实合理）
        array       []
        object      从空标签生成的 SimpleXML 对象（即其他所有对象都会被转换成 true）
        callable    无（即所有 callable 类型的值都会转换成 true）
        resource    无（即所有资源类型的值都会被转换成 true）
        null        null
