伪类型（pseudo-types） 是 PHP 文档中在指示参数（或者返回值）的类型和值时可以使用的关键字。
请注意伪类型不是 PHP 语言的原始事物，所以在自定义函数中不要使用伪类型做类型提示。

mixed
    表示一个参数可以接受多种不同的（但不一定是所有的）类型。
number
    表示一个参数可以是整型或者浮点型。
callback
    在 PHP 5.4 引入 callable 类型之前 PHP 文档中使用 callback 伪类型，二者含义完全相同。
    个人补充：所以以后应当使用 callable 。
array|object
    表示一个参数可以是数组或者对象。
void
    void 作为返回值是表示返回值是无用的；void 作为参数表示函数不接受任何参数。
    从 PHP 7.1.0 开始，函数的返回类型提示可以接受 void 。
    个人补充：即使函数没有指定返回值，也会返回 null ，所以 void 作为返回值时不表示“没有返回值”而表示“返回值是无用的”。

...
    在函数原型中，$... 表示等等的意思。
    当一个函数可以接受任意个参数时使用此变量名。
