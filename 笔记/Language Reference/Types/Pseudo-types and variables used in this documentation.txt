伪类型（pseudo-types） 是 PHP 文档中用于指示参数可以使用的类型和值的关键字。
请注意伪类型不是 PHP 语言的原始事务，所以在自定义函数中不要使用伪类型做类型提示。

mixed
    表示一个参数可以接受多种不同的（但不一定是所有的）类型。
number
    表示一个参数可以是整型或者浮点型。
callback
    PHP 文档中在 PHP 5.4 引入 callable 类型之前使用 callback 伪类型，二者含义完全相同。
    （个人补充）以后应当使用 callable 更好一些。
array|object
    表示一个参数可以是数组或者对象。
void
    void 作为返回值是表示返回值是无用的；void 作为参数表示函数不接受任何参数。
    从 PHP 7.1.0 开始，函数的返回类型提示可以接受 void 。

...
    在函数原型中，$... 表示等等的意思。
    当一个函数可以接受任意个参数时使用此变量名。
    （个人补充）... 是一个伪变量，不是伪类型。
