PHP 为运行它的脚本提供了大量的 预定义常量 。然而，这些常量中有许多是由各种扩展创建的，并且仅当这些扩展可用时才会出现，要么通过动态加载，要么
由于被编译进 PHP 中了。

PHP 提供了 9 个魔术常量，它们的值会根据使用的位置而变化。比如，__LINE__ 的值取决于脚本中使用它所在的行。所有这些魔术常量都是编译时进行解析
的，不像常规常量是在运行时解析的（个人补充：现在 const 关键字声明常量也是编译时进行的）。这些特殊的常量都是大小写不敏感的。

几个魔术 PHP 常量
    名称                 描述
    __LINE__            文件中的当前行号。
    __FILE__	        文件的完整路径和文件名。如果用在被包含文件中，则返回被包含的文件名。
    __DIR__	            文件所在的目录。如果用在被包括文件中，则返回被包含的文件所在的目录。
                        它等价于 dirname(__FILE__)。除非是根目录，否则目录中名不包括末尾的斜杠。
    __FUNCTION__	    函数的名称，如果是匿名函数则返回 {closure} 。
    __CLASS__	        类的名称。类名包括其被声明的命名空间（例如 Foo\Bar）。
                        注意自 PHP 5.4 起 __CLASS__ 对 trait 也起作用。当用在 trait 方法中时，__CLASS__ 是调用 trait 方法的类的
                        名字。
    __TRAIT__	        Trait 的名字。Trait 名包括其被声明的命名空间（例如 Foo\Bar）。
    __METHOD__	        类的方法名。
    __NAMESPACE__	    当前命名空间的名称。
    ClassName::class	完整限定的类名。
另请查阅 get_class()， get_object_vars()， file_exists() 和 function_exists() 。
