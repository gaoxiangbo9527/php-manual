if 结构是许多语言最重要的特性之一，包括 PHP 。它允许有条件地执行代码片段。PHP 的 if 结构类似于 C 语言：
    if (expr)
      statement

正如 表达式一节 所述，表达式的值为其布尔值。如果表达式的求值结果为 TRUE，PHP 将执行语句，如果求值结果为 FALSE —— 它将忽略该语句。关于什么
值会被求值为 FALSE 的更多信息可以在“转换为布尔值”一节中找到。

如果 $a 大于 $b ，下面的例子将会显示 a is bigger than b ：
    <?php
    if ($a > $b)
      echo "a is bigger than b";
    ?>

通常，你会希望有多个语句被有条件地执行。当然，没有必要用 if 子句包装每条语句。相反，你可以将几条语句分组到一个语句组中。例如，如果 $a 大于
$b，那么这段代码将会显示 a is bigger than b ，同时会将 $a 的值赋值到 $b 中。
    <?php
    if ($a > $b) {
      echo "a is bigger than b";
      $b = $a;
    }
    ?>

if 语句可以无限地嵌套在其他的 if 语句中，它为有条件的执行你程序的各个部分提供了完整的灵活性。
