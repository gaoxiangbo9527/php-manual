描述
    header_remove ([ string $name ] ) : void

    移除之前使用 header() 设置的一个 HTTP 头部。

参数
    name
        将要被移除的头部名称。

        注意：这个参数是大小写不敏感的。

        个人补充：如果提供了该参数，那么会移除所有名称为该参数的头部，如果有多个的话。

返回值
    没有返回值。

注意事项
    小心 这个函数将移除 PHP 设置的所有头部，包括 cookie、session 和 X-Powered-By 头部。个人补充：会移除所有由 PHP 设置的头部，但注意非
    PHP 设置的头部不会被移除。

    注意：只有使用的 SAPI 支持头部时，才可以访问和输出头部。
