描述
    hash_equals ( string $known_string , string $user_string ) : bool

    比较两个字符粗，不论它们是否相等，都会使用相同的时间。

    这个功能应该用来减轻计时攻击；例如，在测试 crypt() 的密码散列时。

参数
    known_string
        要拿去比较的已知长度的字符串。
    user_string
        用户提供的字符串。

返回值
    当两个字符串相等时返回 TRUE，否则返回 FALSE 。

错误/异常
    当提供的任何一个参数不是一个字符串时，会产生一个 E_WARNING 消息。

注意事项
    注意：两个参数必须具有相同的长度才能成功地进行比较。当提供不同长度的参数时，会立即返回 FALSE，并且在发生计时攻击时已知字符串的长度可能会
    泄漏。

    注意：将用户提供的字符串作为第二个参数而不是第一个参数是很重要的。
