会话支持在 PHP 中是默认启用的。如果你不想构建具有会话支持的 PHP，你应该指定要配置的 --disable-session 选项。要对会话存储使用共享内存分配
（mm），请配置 PHP --with-mm[=DIR]。

Windows 版本的 PHP 内置的支持这个扩展。你不需要去加载任何额外的扩展以使用这些函数。

注意：默认情况下，与特定会话相关的所有数据将存储在由 session.save_path INI 选项指定的目录下的某个文件中。将为每个会话创建一个文件（无论是
否有数据与该会话关联）。这是由于每打开一个会话就会建立一个文件，即使没有数据会写入到该文件中。请注意，这种行为是与文件系统一起工作的限制的一个
副作用，并且一个自定义的会话处理程序（例如使用一个数据库的会话处理程序）可能不会跟踪不存储数据的会话。
