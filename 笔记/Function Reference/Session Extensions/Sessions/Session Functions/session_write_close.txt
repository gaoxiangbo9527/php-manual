描述
    session_write_close ( void ) : bool

    结束当前的会话并且存储会话数据。

    会话数据通常会在脚本终止后被存储并且不需要去调用 session_write_close()，但由于会话数据被锁定以防止并发写入，因此在任何时候只能有一个脚
    本在某个会话上进行操作。当把框架集和会话一起使用时，由于这种锁定，你将体验到框架会一个接一个地加载。你可以在对会话变量的所有更改完成后立即
    结束会话，从而减少加载所有框架所需的时间。

返回值
    成功时返回 TRUE，失败时返回 FALSE。
