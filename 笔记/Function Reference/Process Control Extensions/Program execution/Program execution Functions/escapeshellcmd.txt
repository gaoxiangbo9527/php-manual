描述
    escapeshellcmd ( string $command ) : string

    escapeshellcmd() 转义一个字符串中可能用于欺骗 shell 命令去执行任意命令的任何字符。应该使用这个函数去确保任何来自用户输入的数据在传递
    给 exec() 或 system() 函数或者 反引号操作符 前被转义。

    以下字符前会有反斜杠：&#;`|*?~<>^()[]{}$\, \x0A 和 \xFF 。' 和 " 只有在没有成对出现的情况下才被转义。在 Windows 上，所有这些字
    符以及 % 和 ! 的前面会有一个插入符号（^）。

参数
    command
        将被转义的命令。

返回值
    转义后的字符串。

示例
    警告 escapeshellcmd() 应该用在整个命令字符串上，并且它仍然允许攻击者传递任意数量的参数。要转义单个参数，则应该使用
    escapeshellarg() 。
