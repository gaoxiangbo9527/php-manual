描述
    shell_exec ( string $cmd ) : string

    该函数与 反引号操作符 相同。

    注意：在 Windows 上，底层管道是以文本模式打开的，对于二进制的输出，该模式可能会导致函数失败。在这种情况下应考虑使用 popen() 。

参数
    cmd
        将被执行的命令。

返回值
    被执行的命令的输出。如果发生错误或者命令没有产生输出时返回 NULL 。

    注意：因为在发生错误时或者命令没有产生输出时都会返回 NULL ，所以使用该函数无法判断执行是否失败。如果需要获取程序的退出码，应当使用
    exec() 。

注意事项
    注意：当 PHP 运行在 安全模式 下时，该函数被禁用。
