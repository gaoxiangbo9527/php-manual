描述
    escapeshellarg ( string $arg ) : string

    escapeshellarg() 在一个字符串周围添加单引号并且引用/转译任何现有的单引号，允许你直接将一个字符串传递给 shell 函数并将其作为单个安全的
    参数对待。这个函数应该用于把来自用户输入的单个参数转义给 shell 函数。这些 shell 函数包括 exec()、system() 和 反引号操作符 。

    在 Windows 中，escapeshellarg() 用空格替换百分号、感叹号（延迟变量替换）和双引号，并在字符串周围添加双引号。

参数
    arg
        将被转义的参数。

返回值
    转义后的字符串。
