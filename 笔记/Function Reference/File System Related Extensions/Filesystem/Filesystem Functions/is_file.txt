描述
    is_file ( string $filename ) : bool

    告知给定的文件是否是一个常规的文件。

参数
    filename
        文件的路径。

返回值
    如果文件名存在且是一个常规的文件，返回 TRUE，否则返回 FALSE 。

    注意：因为 PHP 的整型是有符号的而且许多平台使用 32 位整数，所以对于大于 2GB 的文件，一些文件系统函数可能会返回意料之外的结果。

错误/异常
    失败时，产生一个 E_WARNING 。

注意事项
    注意：这个函数的结果是被缓存的。更多细节请参见 clearstatcache() 。

    提示 从 PHP 5.0.0 开始，这个函数还可以与一些 URL 包装器一起使用。请参考 支持的协议和包装器 ，以确定哪些包装器支持 stat() 系列的功能。
