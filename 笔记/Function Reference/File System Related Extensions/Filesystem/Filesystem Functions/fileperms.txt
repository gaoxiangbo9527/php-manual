描述
    fileperms ( string $filename ) : int

    获取给定文件的权限。

参数
    filename
        文件的路径。

返回值
    以数字模式返回文件的模式。此模式的较低位与 chmod() 所期望的权限相同，然而在大多数平台上，返回值还将包含通过 filename 给定的文件的类型
    信息。下面的示例演示了在 POSIX 系统上，包括 Linux 和 macOS，如何测试特定权限和文件类型的返回值。

    对于本地文件，特定的返回值是 C 库的 stat() 函数返回的结构体的 st_mode 成员。具体设置的位因平台而异，如果需要解析返回值的非权限位，建议
    查阅特定平台的文档。

错误/异常
    失败时，产生一个 E_WARNING 。

注意事项
    注意：这个函数的结果是被缓存的。更多细节请参见 clearstatcache() 。

    提示 从 PHP 5.0.0 开始，这个函数还可以与一些 URL 包装器一起使用。请参考 支持的协议和包装器 ，以确定哪些包装器支持 stat() 系列的功能。
