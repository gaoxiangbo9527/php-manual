描述
    fgetc ( resource $handle ) : string

    从给定的文件指针获取一个字符串。

参数
    handle
        文件指针必须是有效的，并且必须指向一个由 fopen() 或 fsockopen() 成功打开的文件（并且还没有被 fclose() 关闭）。

返回值
    返回一个从 handle 指向的文件中读取到的、包含单个字符的字符串。遇到 EOF 则返回 FALSE 。

    警告 此函数可能返回布尔值 FALSE，但也可能返回计算结果为 FALSE 的非布尔值。请阅读 布尔值 部分以获得更多信息。使用 === 操作符测试此函数
    的返回值。

注意事项
    注意：此函数是二进制安全的。
