描述
    fread ( resource $handle , int $length ) : string

    fread() 从由 handle 所引用的文件指针中最多读取 length 个字节。当满足下列条件之一时就会停止读取：
        已读取了 length 个字节
        到达 EOF（文件末尾）
        一个数据包变得可用或发生套接字超时（对于网络流）
        如果流读取被缓冲并且它不代表一个普通文件，最多一次读取等于块大小的字节数（通常为 8192）；取决于以前缓冲的数据，返回数据的大小可能大
        于块的大小。

参数
    handle
        通常使用 fopen() 创建的一个文件系统指针资源。
    length
        最多读取 length 数量的字节。

返回值
    返回读取的字符串或者在失败时返回 FALSE 。

示例
    警告 在区分二进制文件和文本文件的系统上（如 Windows），文件必须以在 fopen() mode 参数中包含 'b' 的方式打开。

    警告 当从一个非常规的本地文件中读取时，如读取 远程文件 返回的流或从 popen() 和 fsockopen() 读取时，在一个数据包变得可用后读取将停止。

注意事项
    注意：如果你只是想获取一个文件的内容到一个字符串，可以使用 file_get_contents()，因为它比上面的代码拥有更好的性能。

    注意：应注意 fread() 会从文件指针的当前位置读取。使用 ftell() 来查找指针的当前位置并使用 rewind() 来重置指针的位置。
