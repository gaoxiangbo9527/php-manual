描述
    is_dir ( string $filename ) : bool

    告知给定的文件名是否是一个目录。

参数
    filename
        文件的路径。如果 filename 是一个相对的文件名，那么它将相对于当前工作目录进行检查。如果 filename 是一个符号链接或硬链接，那么将解
        析并检查该链接。如果你启用了 安全模式 或者 open_basedir ，则可能会应用更进一步的限制。

返回值
    如果文件名存在并且是一个目录，返回 TRUE，否则返回 FALSE 。

错误/异常
    失败时，产生一个 E_WARNING 。

注意事项
    注意：这个函数的结果是被缓存的。更多细节请参见 clearstatcache() 。

    提示 从 PHP 5.0.0 开始，这个函数还可以与一些 URL 包装器一起使用。请参考 支持的协议和包装器 ，以确定哪些包装器支持 stat() 系列的功能。
