描述
    ftell ( resource $handle ) : int

    返回由 handle 所引用的文件指针的位置。

参数
    handle
        文件指针必须是有效的，并且必须指向一个由 fopen() 或 popen() 成功打开的文件。对于仅添加的流（用'a'标志打开），ftell() 会给出未定
        义的结果。

返回值
    以一个整数返回由 handle 所引用的文件指针的位置。即，它在文件流中的偏移量。

    如果发生一个错误，则返回 FALSE 。

    注意：因为 PHP 的整数类型是有符号的并且许多平台使用 32 位整数，所以一些文件系统函数可能会对大于 2GB 的文件返回意料之外的结果。
