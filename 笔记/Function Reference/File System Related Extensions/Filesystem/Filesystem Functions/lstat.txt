描述
    lstat ( string $filename ) : array

    收集由 filename 指定的文件或符号链接的统计信息。

参数
    filename
        一个文件或符号链接的路径。

返回值
    关于 lstat() 返回的数组结构的信息，请参阅 stat() 的手册页。这个函数与 stat() 函数相同，除了如果 filename 参数是一个符号链接，则返回
    符号链接的状态，而不是符号链接所指向的文件的状态。

错误/异常
    失败时，产生一个 E_WARNING 。

注意事项
    注意：这个函数的结果是被缓存的。更多细节请参见 clearstatcache() 。

    提示 从 PHP 5.0.0 开始，这个函数还可以与一些 URL 包装器一起使用。请参考 支持的协议和包装器 ，以确定哪些包装器支持 stat() 系列的功能。
