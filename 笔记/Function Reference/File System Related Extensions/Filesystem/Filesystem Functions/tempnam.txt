描述
    tempnam ( string $dir , string $prefix ) : string

    在指定的目录中创建一个具有唯一文件名、访问权限设置为 0600 的文件。如果该目录不存在或者不可写，tempnam() 可能会在系统的临时目录中生成一
    个文件，并返回这个文件的完整路径，包括它的名称。

参数
    dir
        临时文件将被创建在的目录。
    prefix
        被生成的临时文件的前缀。

        注意：只使用前缀的前 63 个字符。Windows 甚至只使用前缀的前 3 个字符。

返回值
    返回新的临时文件名（带路径），失败时返回 FALSE 。

注意事项
    注意：如果 PHP 不能在指定的 dir 参数中创建一个文件，它会回退到系统默认（的临时目录）。在 NTFS 上，如果指定的 dir 包含超过 65534 个文
    件，也会发生这种情况。
