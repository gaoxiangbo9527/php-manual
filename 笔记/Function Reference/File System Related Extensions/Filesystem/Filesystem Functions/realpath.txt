描述
    realpath ( string $path ) : string

    realpath() 会展开 path 中的所有符号链接并且会解析 path 中的到 /./，/../ 和多余的 / 字符的引用，并返回规范化后的绝对路径。

参数
    path
        被检查的路径。

        注意：虽然必须提供一个路径，但值可以是空字符串。在这种情况下，该值被解释为当前目录。

返回值
    成功时返回规范化后的绝对路径名。结果路径将不会有符号链接，/./ 或 /../ 组件。尾部的分隔符，例如 \ 和 /，也将被移除。

    失败时，例如，如果该文件不存在，将返回 FALSE 。

    注意：运行中的脚本对于层次结构中的所有目录都必须具有可执行权限，否则 realpath() 将返回 FALSE 。

    注意：对于不区分大小写的文件系统，realpath() 可能规范化字符大小写，也可能不规范化。

    注意：函数 realpath() 不适用于 Phar 中的文件，因为这样的路径将是一个虚拟的路径，而不是一个真实的路径。

    注意：因为 PHP 的整型是有符号的并且许多平台使用 32 位整型，对于那些大于 2GB 的文件，一些文件系统函数可能会返回出乎意料的结果。
