描述
    readdir ([ resource $dir_handle ] ) : string

    返回目录中下一个条目的名称。会按照条目在文件系统中存储的顺序返回。

参数
    dir_handle
        之前通过 opendir() 打开的目录句柄资源。如果没有指定目录句柄，就假设是由 opendir() 打开的最后一个链接。

返回值
    成功时返回条目的名称，失败时返回 FALSE 。

    警告 这个函数可能会布尔值 FALSE ，但是也可能会返回一个求值为 FALSE 的非布尔值。更多信息请阅读 布尔值 部分。请使用 === 操作符去测试这
    个函数的返回值。
