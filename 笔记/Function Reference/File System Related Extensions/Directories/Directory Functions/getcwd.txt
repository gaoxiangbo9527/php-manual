描述
    getcwd ( void ) : string

    获取当前工作目录。

返回值
    成功时返回当前工作目录，失败时返回 FALSE 。

    在一些 Unix 变种上，如果上级目录中的任何一个没有设置可读权限或搜索模式，getcwd() 将返回 FALSE，即使当前目录设置了可读权限或搜索模式。
    关于模式和权限的更多信息，请参考 chmod() 。

    小心 如果 PHP 解释器是在启用 ZTS （Zend 线程安全）的情况下构建的，那么 getcwd() 返回的当前工作目录可能与操作系统接口返回的当前工作目
    录不同。依赖于当前工作目录的外部库（通过 FFI 调用）将受到影响。
