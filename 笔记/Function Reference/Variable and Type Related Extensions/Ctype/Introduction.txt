由此扩展提供的函数会根据当前的区域（另请参阅 setlocale()）去检查一个字符或字符串是否属于某种字符类别。

当使用一个整型参数调用这些函数时，它们的行为与 ctype.h 中对应的 C 函数完全相同。这意味着如果你传递一个小于 256 的整数，该函数将使用它的
ASCII 值来查看它是否符合指定的范围（数字在 0x30-0x39 中）。如果数字介于 -128 和 -1 之间，那么将加上 256，然后再对其进行检查。

当使用一个字符串参数调用这些函数时，它们会检查字符串中的每一个字符并且只有当字符串中的每一个字符都符合所请求的标准时才会返回 TRUE 。当使用一
个空字符串调用这些函数时，在 PHP < 5.1 中总是返回 TRUE，而从 5.1 开始总是返回 FALSE 。

传递字符串或整数以外的任何东西都将立刻返回 FALSE 。

应该注意的是，ctype 函数总是比正则表达式好，甚至比一些等效的 “str_*” 和 “is_*” 函数好。这是因为 ctype 使用的是一个原生的 C 库，因此处理
速度快的非常多。

注意：这些函数与 Python 的 “ctypes” 库没有任何关系。此扩展名源自于 C 头文件 ctype.h ，与它们等价的 C 函数就定义在这个头文件中。这个扩展
也早于 Python 的 “ctypes”，所以任何由这个命名引起的混乱都不是我们的错…
