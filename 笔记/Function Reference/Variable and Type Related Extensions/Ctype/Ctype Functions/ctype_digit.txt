描述
    ctype_digit ( string $text ) : bool

    检查在 text 提供的字符串中的所有字符是否都是数字。

参数
    text
        被测试的字符串。

返回值
    如果 text 中的每个字符都是一个十进制数字，那么就返回 TRUE ，否则返回 FALSE 。

变更日志
    版本	    描述
    5.1.0	在 PHP 5.1.0 前，当 text 是一个空字符串时，此函数会返回 TRUE 。

注意事项
    注意：这个函数需要一个字符串才是有用的，因此传入一个整数可能不会返回预期的结果。然而，也要注意 HTML 表单将生成数值化字符串而不是整数。请
    参阅手册的 类型 部分。

    注意：如果提供了一个在 -128 到 255 之间的整数，它会被解释为单个字符的 ASCII 值（为了允许字符位于扩展的 ASCII 范围内，需要为负数加上
    256 ）。任何其他整数都会被解释为包含该整数的十进制数字的字符串。
