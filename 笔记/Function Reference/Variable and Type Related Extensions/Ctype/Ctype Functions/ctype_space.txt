描述
    ctype_space ( string $text ) : bool

    检查在 text 提供的字符串中的所有字符是否都创建了空白空间。

参数
    text
        被测试的字符串。

返回值
    如果 text 中的每个字符都创建了一些空白空间，那么就返回 TRUE ，否则返回 FALSE 。除了空格字符，它还包括制表符，垂直制表符，换行符，回车
    符和换页符。

注意事项
    注意：如果提供了一个在 -128 到 255 之间的整数，它会被解释为单个字符的 ASCII 值（为了允许字符位于扩展的 ASCII 范围内，需要为负数加上
    256 ）。任何其他整数都会被解释为包含该整数的十进制数字的字符串。
