描述
    ctype_alpha ( string $text ) : bool

    检查在 text 提供的字符串中的所有字符是否都是字母。在标准的 C 区域中，字母只是 [A-Za-z]，并且如果 $text 只是一个字符，那么
    ctype_alpha() 相当于（ctype_upper($text) || ctype_lower($text)），但其他语言的字母被认为既不是大写也不是小写。

参数
    text
        被测试的字符串。

返回值
    如果 text 中的每个字符都是一个来自当前区域中的字母，那么就返回 TRUE ，否则返回 FALSE 。

注意事项
    注意：如果提供了一个在 -128 到 255 之间的整数，它会被解释为单个字符的 ASCII 值（为了允许字符位于扩展的 ASCII 范围内，需要为负数加上
    256 ）。任何其他整数都会被解释为包含该整数的十进制数字的字符串。
