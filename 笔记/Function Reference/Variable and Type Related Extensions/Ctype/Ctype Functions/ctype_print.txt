描述
    ctype_print ( string $text ) : bool

    检查在 text 提供的字符串中的所有字符是否都是可打印的。

参数
    text
        被测试的字符串。

返回值
    如果 text 中的每个字符都会创建实际的输出（包括空格），那么就返回 TRUE 。否则，如果 text 包含控制字符或者不会有任何输出或控制功能的字符
    的话，则返回 FALSE 。

注意事项
    注意：如果提供了一个在 -128 到 255 之间的整数，它会被解释为单个字符的 ASCII 值（为了允许字符位于扩展的 ASCII 范围内，需要为负数加上
    256 ）。任何其他整数都会被解释为包含该整数的十进制数字的字符串。
