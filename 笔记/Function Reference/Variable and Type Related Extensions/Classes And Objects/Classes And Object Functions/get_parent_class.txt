描述
    get_parent_class ([ mixed $object ] ) : string
    获取对象或者类的父类的名称。
参数
    object
        被测试的对象或类的名称。如果从对象的方法中调用该函数，那么该参数是可选的。
返回值
    如果 object 是一个类的实例或者名称，则返回它的父类的名称。
    注意：如果 object 没有一个父类或者指定的类不存在，会返回 FALSE 。
    如果在对象外没有使用参数调用该函数，则返回 FALSE 。
变更日志
    版本         描述
    5.1.0	    如果在对象外没有使用参数调用该函数，过去会返回 NULL 和一个警告，但是现在会返回 FALSE 。
