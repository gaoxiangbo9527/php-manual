描述
    prev ( array &$array ) : mixed
    倒回内部数组指针。
    prev() 的行为非常类似于 next()，除了它会倒退内部数组指针一个位置而不是向前推进一个位置。

参数
    array
        输入数组。

返回值
    返回内部数组指针指向的前一个位置的数组值，如果没有更多的元素，则返回 FALSE 。

    警告 此函数可以返回布尔值 FALSE，但也可以返回一个求值为 FALSE 的非布尔值。请阅读 布尔值 章节以获得更多信息。请使用 === 操作符测试此函
    数的返回值。

注意事项
    注意：一个数组的开头无法与一个布尔型 FALSE 元素进行区分。为了进行区分，请检查 prev() 元素的 key() 是否不为 NULL 。

个人补充：这些用于遍历的数组函数实现的不算好，除了 key() ，其他的几个函数都包含了多余的意图或返回值，使用它们时注意不要用这些多余的意图或返
回值，否则麻烦会很多。key() 很重要，因为除了获取键以外，也相当于 valid() 。
