描述
    compact ( mixed $var_name , mixed ...$var_names ) : array

    创建一个包含变量和它们的值的数组。

    对于这些参数，compact() 在当前符号表中查找具有该名称的变量，并将其添加到输出数组中，变量名成为键并且变量的内容成为键的值。简而言之，它的
    作用与 extract() 相反。

    注意：在 PHP 7.3 之前，任何没有设置的字符串将会被安静的跳过。

参数
    var_name
    var_names
        compact() 接受一份可变数量的参数。每个参数可以要么是一个包含了变量名的字符串，要么是一个变量名的数组。数组可以在它内部包含其他的变
        量名数组，compact() 会递归的处理它。

返回值
    返回所有变量都会添加到它的输出数组。

错误/异常
    如果给定的字符串引用了一个未设置的变量，compact() 发出一个 E_NOTICE 级别的错误。

更新日志
    版本    描述
    7.3.0   如果给定的字符串引用了一个未设置的变量，compact() 现在会发出一个 E_NOTICE 级别的错误。之前，这样的字符串会被安静的跳过。

注意事项
    注意：因为 可变变量 在函数中不能与 PHP 的超全局数组一起使用，超全局数组不能被传递到 compact() 中。
