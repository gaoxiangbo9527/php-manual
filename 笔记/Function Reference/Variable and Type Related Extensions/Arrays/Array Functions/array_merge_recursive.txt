描述
    array_merge_recursive ( array ...$arrays ) : array

    把一个或多个数组的元素合并到一起，从而一个数组的值会追加到前一个数组的末尾。它返回结果数组。

    如果输入数组拥有相同的字符串键，那么这些键的值会被合并到同一个数组中，并且这会递归的完成，从而如果这些值中的某一个自身就是一个数组，该函数
    也会将其与另一个数组中相应的条目合并。然而，如果数组包含数字索引，后一个值不会覆盖原始的值，而将会被追加。

参数
    arrays
        列出要递归合并的数组的变量。

返回值
    把参数合并到一起后得到的一个数组值。如果没有使用任何参数进行调用，则返回一个空数组。
