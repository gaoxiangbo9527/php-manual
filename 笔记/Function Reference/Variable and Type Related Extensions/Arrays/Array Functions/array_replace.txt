描述
    array_replace ( array $array [, array $replacements ] ) : array

    array_replace() 将 array 的值替换为后面每个数组中具有相同键的值。如果一个来自第一个数组的键存在于第二个数组中，那么它的值将会被来自第
    二个数组的值所替换。如果键存在于第二个数组中，并且不存在于第一个数组中，那么将会在第一个数组中创建该键。如果一个键只存在于第一个数组中，
    那么它将原样保留。如果传递了多个数组以进行替换，它们将会被按顺序处理，后面的数组会覆盖前面的值。

    array_replace() 不是递归的：它将用第二个数组中的任何类型替换第一个数组中的值。

参数
    array
        其中的元素要被替换的数组。
    replacements
        要从中提取元素的数组。来自后面数组的值将会覆盖前面的值。

返回值
    返回一个数组，如果发生一个错误，则为 NULL。
