描述
    array ([ mixed $... ] ) : array
    创建一个数组。请阅读 array 类型的章节去了解更多关于数组是什么的信息。

参数
    ...
        语法 "index => values"，由逗号分隔，定义了索引和值。索引的类型可以使字符串或者整数。当索引被省略时，会自动生成一个整数索引，从 0
        开始。如果索引是一个整数，那么生成的下一个索引将会是当前最大的整数索引 + 1 。注意当定义了两个相同的索引时，后面的会覆盖前面的。

        在最后定义的数组记录的后面有一个尾随的逗号，虽然不常见，但确实是一种有效的语法。

返回值
    返回这些参数组成的一个数组。参数可以使用 => 操作符去指定一个索引。

注意事项
    注意
        array() 是一个用于表示数组字面量的语言结构，而不是一个常规的函数。
