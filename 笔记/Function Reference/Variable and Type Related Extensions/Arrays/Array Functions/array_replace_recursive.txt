描述
    array_replace_recursive ( array $array [, array $replacements ] ) : array

    array_replace_recursive() 使用来自后面所有数组的相同的值替换 array 中的值。如果一个来自第一个数组的键存在于第二个数组中，那么它的值
    将会被来自第二个数组的值所替换。如果键存在于第二个数组中，并且不存在于第一个数组中，那么将会在第一个数组中创建该键。如果一个键只存在于第一
    个数组中，那么它将原样保留。如果传递了多个数组以进行替换，它们将会被按顺序处理，后面的数组会覆盖前面的值。

    array_replace_recursive() 是递归的：它将递归进数组中，并对内部值应用相同的处理。

    当第一个数组中的值是标量时，它将被第二个数组中的值所替换，该值可能是标量或者数组。当第一个数组和第二个数组中的值都是数组时，
    array_replace_recursive() 将递归的替换它们对应的值。

参数
    array
        其中的元素要被替换的数组。
    replacements
        要从中提取元素的数组。来自后面数组的值将会覆盖前面的值。

返回值
    返回一个数组，如果发生一个错误，则为 NULL。
