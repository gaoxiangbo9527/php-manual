描述
    next ( array &$array ) : mixed
    next() 的行为类似于 current()，只有一处不同。它会在返回元素值之前把内部数组指针向前推进一个位置。那就意味着它会返回下一个数组值并将内
    部数组指针向前进 1。

参数
    array
        将要被影响的数组。

返回值
    返回内部数组指针指向的下一个位置的数组值，如果没有更多的元素，则返回 FALSE 。

    警告 此函数可以返回布尔值 FALSE，但也可以返回一个求值为 FALSE 的非布尔值。请阅读 布尔值 章节以获得更多信息。请使用 === 操作符测试此函
    数的返回值。

注意事项
    注意：一个数组的末尾无法与一个布尔型 FALSE 元素进行区分。要正确的遍历一个可能包含了 FALSE 元素的数组，请参考 foreach() 函数。要想仍然
    使用 next() 并正确地检查是否到达了数组的末尾，请验证 key() 是否为空。

个人补充：这些用于遍历的数组函数实现的不算好，除了 key() ，其他的几个函数都包含了多余的意图或返回值，使用它们时注意不要用这些多余的意图或返
回值，否则麻烦会很多。key() 很重要，因为除了获取键以外，也相当于 valid() 。
