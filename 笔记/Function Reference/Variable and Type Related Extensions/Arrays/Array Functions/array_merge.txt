描述
    array_merge ( array ...$arrays ) : array

    把一个或多个数组的元素合并到一起，从而一个数组的值会追加到前一个数组的末尾。它返回结果数组。

    如果输入数组拥有相同的字符串键，那么该键的后一个值将覆盖前一个。然而，如果数组包含数字索引，后一个值不会覆盖原始的值，而将会被追加。

    输入数组中具有数字键的值在结果数组中将以从零开始的递增键重新编号。

参数
    arrays
        列出要合并的数组的变量。

返回值
    返回结果数组。如果不使用任何参数调用，将返回一个空数组。
