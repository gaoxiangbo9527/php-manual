描述
    usort ( array &$array , callable $value_compare_func ) : bool

    这个函数将根据数组的值对数组进行排序，使用一个用户提供的比较函数。如果希望排序的数组需要根据一些非平凡的条件进行排序，那么应该使用此函数。

    注意：如果两个成员相等，它们在排序后的数组中的相对顺序是未定义的。

    注意：这个函数给 array 中的元素赋予了新的键。它将移除任何可能已经分配的现有键，而不仅仅是重新排序键。

参数
    array
        输入数组。
    value_compare_func
        比较函数必须返回一个小于，等于或者大于 0 的整数，如果第一个参数被认为是对应的小于，等于或者大于第二个参数。注意，在 PHP 7.0.0 之前，
        这个整数必须在范围 -2147483648 到 2147483647 中。

        callback ( mixed $a, mixed $b ) : int

        小心
            从比较函数返回一个非整型值，例如浮点数，将会在内部强制的把回调的返回值转换为整型。因此，像 0.99 和 0.1 这样的值都将被转换成一个
            值为 0 的整数，这样的值会被当作相等。

返回值
    成功返回 TRUE，失败返回 FALSE。
