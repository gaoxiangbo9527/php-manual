描述
    array_rand ( array $array [, int $num = 1 ] ) : mixed

    从数组中挑选一个或多个随机条目，并返回随机记录的键。它使用一个伪随机数生成器，该生成器并不适合加密目的。

参数
    array
        输入数组。
    num
        指定应该挑选多少条目。

返回值
    当只挑选一个条目时，array_rand() 返回这个随机条目的键。否则，会为多个随机条目返回其键的数组。这样做是为了从数组中选择随机键和随机值。
    如果多个键被返回，将按照它们在原数组中出现的顺序被返回。尝试挑选比数组自身更多的元素将导致一个 E_WARNING 级别的错误，并且将返回 NULL。

更新日志
    版本     描述
    7.1.0	内部的随机算法已经改为使用 Mersenne Twister 随机数生成器而不是 libc 的 rand 函数。
