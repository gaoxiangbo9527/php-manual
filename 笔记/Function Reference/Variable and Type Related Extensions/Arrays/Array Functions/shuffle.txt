描述
    shuffle ( array &$array ) : bool

    这个函数对数组进行洗牌（随机数组中元素的顺序）。它使用一个伪随机数生成器，这个生成器不适合加密目的。

参数
    array
        数组。

返回值
    成功返回 TRUE，失败返回 FALSE。

更新日志
    版本     描述
    7.1.0	内部的随机算法已经改为使用 Mersenne Twister 随机数生成器而不是 libc 的 rand 函数。

注意事项
    注意：这个函数给 array 中的元素赋予了新的键。它将移除任何可能已经分配的现有键，而不仅仅是重新排序键。
