描述
    current ( array $array ) : mixed
    返回数组的当前元素。
    每个数组都有一个到它的当前元素的一个内部指针，该指针会初始化为插入到该数组中的第一个元素。

参数
    array
        数组。

返回值
    current() 函数会简单的返回当前被内部指针指向的数组元素的值。它不会以任何方式去移动该指针。如果内部指针指向元素列表末尾以外，或者数组为
    空，那么 current() 返回 FALSE 。

    警告 此函数可以返回布尔值 FALSE，但也可以返回一个求值为 FALSE 的非布尔值。请阅读 布尔值 章节以获得更多信息。请使用 === 操作符测试此函
    数的返回值。

变更日志
    版本        描述
    7.0.0       现在总是按值传递 array 。在这个版本之前，如果可能的话就会按引用传递 array，不可能的话才会按值传递 array 。

注意事项
    注意 数组的末尾和对空数组调用 current() 的结果无法一个布尔的型 FALSE 元素进行区分。想要正确的遍历一个可能会包含 FALSE 元素的数组，请
    参考 foreach() 函数。要想仍然使用 current() 并正确地检查该值是否真的是数组的一个元素，应该检查 current() 元素的 key() 是否与 NULL
    完全不同。

个人补充：这些用于遍历的数组函数实现的不算好，除了 key() ，其他的几个函数都包含了多余的意图或返回值，使用它们时注意不要用这些多余的意图或返
回值，否则麻烦会很多。key() 很重要，因为除了获取键以外，也相当于 valid() 。
