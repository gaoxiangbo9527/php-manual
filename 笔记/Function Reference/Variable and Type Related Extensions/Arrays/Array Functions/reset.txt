描述
    reset ( array &$array ) : mixed
    reset() 会把 array 的内部指针重置到第一个元素并且会返回数组的第一个元素的值。

参数
    array
        输入数组。

返回值
    返回数组的第一个元素的值，如果数组是空的，则返回 FALSE 。

    警告 此函数可以返回布尔值 FALSE，但也可以返回一个求值为 FALSE 的非布尔值。请阅读 布尔值 章节以获得更多信息。请使用 === 操作符测试此函
    数的返回值。

注意事项
    注意：空数组的返回值与第一个元素是布尔值假的数组的返回值是无法区分的。为了正确的检查第一个元素可能是 FALSE 的数组的第一个元素的值，首先
    检查数组的 count()，或者检查 key() 不为 NULL，然后在调用 reset() 。

个人补充：这些用于遍历的数组函数实现的不算好，除了 key() ，其他的几个函数都包含了多余的意图或返回值，使用它们时注意不要用这些多余的意图或返
回值，否则麻烦会很多。key() 很重要，因为除了获取键以外，也相当于 valid() 。
