描述
    func_num_args ( void ) : int
    获取传递给函数的参数的数量。
    该函数可以同 func_get_arg() 和 func_get_args() 联合使用，从而允许用户定义的函数去接受可变数量的参数列表。
返回值
    返回传递给当前用户定义的函数的参数的数量。
变更日志
    版本      描述
    5.3.0    该函数现在可以被用于参数列表中。
    5.3.0    如果该函数是在一个文件的最外层作用域中被调用的，而该文件是通过在调用文件的一个函数中的 include 或 require 来包含的，那么它会生成
             一个警告并返回 -1。
错误和异常
    如果该函数在用户定义的函数的外部被调用，会生成一个警告。
注意
    因为该函数依赖于当前作用域来确定参数细节，在 5.3.0 之前的版本中，它不能用作函数参数。如果这个值必须被传递，应当先将结果赋值给一个变量，然后传递
    这个变量即可。
