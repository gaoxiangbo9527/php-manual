描述
    func_get_arg ( int $arg_num ) : mixed
    从一个用户定义的函数的参数列表中获取特定的参数。
    该函数可以同 func_get_args() 和 func_num_args() 联合使用，从而允许用户定义的函数去接受可变数量的参数列表。
参数
    arg_num
        参数的偏移量。函数的参数是从 0 开始计数的。
返回值
    返回特定的参数，失败时返回 FALSE 。
变更日志
    版本      描述
    5.3.0    该函数现在可以被用于参数列表中。
    5.3.0    如果该函数是在一个文件的最外层作用域中被调用的，而该文件是通过在调用文件的一个函数中的 include 或 require 来包含的，那么它会生成
             一个警告并返回 FALSE 。
错误和异常
    如果该函数在用户定义的函数的外部被调用或者如果 arg_num 大于实际传递的参数数目，会生成一个警告。
注意
    注意
        因为该函数依赖于当前作用域来确定参数细节，在 5.3.0 之前的版本中，它不能用作函数参数。如果这个值必须被传递，应当先将结果赋值给一个变量，然后
        传递这个变量即可。
    注意
        如果参数是按引用传递的，那么对该参数的任何修改也会反馈到该函数返回的值上。从 PHP 7 开始，如果参数是按照值传递的，也会返回当前最新的值。
        （个人补充，PHP 7 这种方式是更合理的方式，PHP 5 的处理方式很容易写出bug）
    注意
        函数数仅返回被传递的参数的一份副本，并且不会考虑默认（即没有实际传递的）参数。
