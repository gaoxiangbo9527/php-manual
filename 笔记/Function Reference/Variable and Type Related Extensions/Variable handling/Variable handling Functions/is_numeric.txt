描述
    bool is_numeric( mixed $var )
    判断给定的变量是否是数值或者数值化字符串。

    数值字符串由可选的符号，任意数量的数字，可选的小数部分和指数部分组成。
    所以使用整型表示法中的十进制表示法和八进制表示法的字符串算作数值化字符串，而二进制表示法和十六进制表示法不算作数值化字符串，即使二进制表示法中只会
    包含一个字母 b 或者 B。

参数
    var
        要评估的变量。
返回值
    如果 var 是数值（整型或者浮点型）或者数值化字符串返回 true ，否者返回 false 。

注意
    从 PHP 7.0.0 开始，使用十六进制表示法的整数字符串不再被认为是数值字符串（个人猜测是因为十六进制中很可能会包含很多的字母）。
