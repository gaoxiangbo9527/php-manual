描述
    is_numeric ( mixed $var ) : bool
    判断给定的变量是否是数值或者数值化字符串。

    数值字符串由可选的符号，任意数量的数字，可选的小数部分和指数部分组成。
    所以使用整型表示法中的十进制表示法和八进制表示法的字符串算作数值化字符串，而二进制表示法和十六进制表示法不算作数值化字符串，即使二进制表
    示法中只会包含一个字母 b 或者 B。

参数
    var
        待求值的变量。

返回值
    如果 var 是数值（整型或者浮点型）或者数值化字符串返回 true ，否者返回 false 。

变更日志
    版本         描述
    7.0.0       使用十六进制表示法的整数字符串（e.g. 0xf4c3b00c）不再被认为是数值字符串，is_numeric() 现在会返回 FALSE 。
