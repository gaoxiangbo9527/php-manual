描述
    intval ( mixed $var [, int $base = 10 ] ) : int
    返回 var 的整型值，转换会使用 base 所指定的进制（默认是 10 进制）。intval() 不应该用于对象上，这么做会产生一个 E_NOTICE 级别的错误，
    并且返回 1 。

参数
    var
        要转换成整型的变量或者标量
    base
        转换所使用的进制。

        注意
            如果 base 的值是 0 ，则实际使用的进制是由 var 参数的格式决定的：
                如果 var 字符串以 0x 或者 0X 开头，则使用十六进制；
                否者，如果 var 字符串以 0b 或者 0B 开头，则使用二进制；
                否者，如果 var 字符串以 0 开头，则使用八进制；
                否者，使用十进制。

返回值
    成功时返回 var 对应的整型值，失败返回 0 。
    字符串可能会返回 0 值，主要取决于字符串最左侧的字符。
    空数组返回 0 ，非空数组返回 1 。
    该函数应该用于对象时会产生一个 E_NOTICE 级别的错误并且会返回 1 。

注意事项
    注意：除非 var 参数是字符串时，否则 base 参数无效。

变更日志
    版本         描述
    5.1.0       当传递给 var 的是一个对象时，会抛出 E_NOTICE 并且返回 1 。

个人补充：其实该函数的转换行为和强制转换 (int) 是一致的。
