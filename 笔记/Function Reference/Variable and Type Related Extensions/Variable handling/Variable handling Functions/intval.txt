描述
    int intval(mixed $var[, int $base = 10])
    返回变量 var 的整型值，基于 base 所指定的进制。
参数
    var
        要转换成整型的变量或者标量
    base
        转换所使用的进制。
        注意
            只有当 var 参数是字符串时，base 参数才有用。

            如果 base 的值是 0 ，则实际使用的进制是由 var 参数的格式决定的：
                如果 var 字符串以 0x 或者 0X 开头，则使用十六进制；
                否者，如果 var 字符串以 0 开头，则使用八进制；
                否者，使用十进制。
返回值
    成功时返回 var 对应的整型值，失败返回 0 。
    字符串可能会返回 0 值，主要取决于字符串最左侧的字符。
    空数组返回 0 ，非空数组返回 1 。
    该函数不应该用于对象，否者会产生一个 E_NOTICE 级别的错误并且会返回 1 。
