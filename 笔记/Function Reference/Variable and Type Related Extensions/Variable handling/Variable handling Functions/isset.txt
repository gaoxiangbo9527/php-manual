描述
    isset ( mixed $var [, mixed $... ] ) : bool
    判断一个变量是否被视为已设置的，这意味着一个变量是已声明的并且不同于 NULL 。
    如果一个变量已经使用 unset() 函数重置，那么它不再被认为是已设置的。
    当检查一个被赋值为 NULL 的变量时，isset() 会返回 FALSE 。也要注意一个空字符（"\0"）是不等同于 PHP NULL 常量的。
    如果提供了多个参数，那么 isset() 仅在所有参数都被认为是已设置的情况下才会返回 TRUE 。求值是从左到右进行的并且一旦遇到一个未设置的变量
    就会停止求值。

参数
    var
        待检查的变量。
    ...
        其他的变量...

返回值
    如果 var 存在并且拥有除 NULL 以外任意的值，则返回 TRUE；否则，返回 FALSE 。

变更日志
    版本         描述
    5.4.0       检查字符串的非数值偏移量现在会返回 FALSE 。

注意事项
    警告 isset() 仅可以和变量一起工作，因此传入任何其他事物会导致一个解析错误。对于检查是否设置了常量，应使用 defined() 函数。

    注意：因为它是一个语言结构，而不是一个函数，所以不能使用 变量函数 调用它。

    注意：当在一个不可访问的对象属性上使用 isset() 时，__isset() 重载方法（解释器钩子）会被调用，如果已存在的话。
