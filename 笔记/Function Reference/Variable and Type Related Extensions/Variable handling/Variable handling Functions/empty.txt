描述
    empty ( mixed $var ) : bool
    判断是否一个变量被认为是空的。
    如果一个变量不存在或者它的值和 FALSE 相等，那么它就被认为是空的。如果变量不存在，empty() 不会生成一个警告。

参数
    var
        被检查的变量。

        注意：
            在 PHP 5.5 以前，empty() 仅支持变量；任何其他事物都会导致一个解析错误。换句话说，这样的代码不会工作：empty(trim($name)) 。
            相反的，应使用 trim($name) == false 。

        如果变量不存在，不会生成警告。 这意味着 empty() 本质上等价于 !isset($var) || $var == false 。

返回值
    如果变量存在并且有一个非空、非 0 值，就返回 FALSE ，否则返回 TRUE 。

    下面的值被认为是空的：
        FALSE
        0
        0.0
        ""
        "0"
        array()
        NULL
        个人补充：按照上面的说法，那么从空标签生成的 SimpleXML 对象应该也会被认为是空的。

变更日志
    版本          描述
    5.5.0       empty() 现在支持表达式，而不再只支持变量。
    5.4.0       检查字符串的非数值偏移量将返回 TRUE 。

注意事项
    注意：由于它是一个语言结构而不是一个函数，所以不能使用 变量函数 调用它。
    注意：当在不可访问的对象属性上使用 empty() 时，__isset() 重载方法会被调用，如果声明了的话。
