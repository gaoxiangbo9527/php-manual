PHP Data Objects（PDO）扩展定义了一个轻量的、一致的接口，用于使用 PHP 访问数据库。每一种实现了 PDO 接口的数据库驱动能够把数据库特定的特性
作为常规的扩展函数公开。注意，你不能仅使用 PDO 扩展本身去执行任何数据库功能；你必须使用一种数据库特定的 PDO 驱动去访问某一个数据库服务器。

PDO 提供了一个“数据访问抽象层”，也就是说，不管你用的是哪种数据库，你都会使用相同的函数去发起查询和获取数据。PDO 没有提供一个“数据库抽象层”；
它不会重写 SQL 或模拟缺少的特性。如果你需要那种能力，你应该使用某个成熟的抽象层。

PHP 5.1 装载了 PDO，并且在 PHP 5.0 中有一个作为 PECL 的 PDO 扩展可用；PDO 需要 PHP 5 核心中的新的 OO 特性，因此不能在早期版本的 PHP
中运行。
