描述
    apcu_store ( string $key , mixed $var [, int $ttl = 0 ] ) : bool
    apcu_store ( array $values [, mixed $unused = NULL [, int $ttl = 0 ]] ) : array
    把一个变量缓存到一个数据存储中。

    注意：不像 PHP 中的许多其他机制，使用 apcu_store() 存储的变量将在请求之间保持（直到值从缓存中删除）。个人补充：这里插这么一句，有点让
    人一时反应不过来。PHP 会在请求结束时销毁所有的资源，所以正常情况下变量不会在请求之间保持，而像 APCu 这种缓存可以改变这种行为。

参数
    key
        使用这个名称存储变量。key 是缓存唯一的，因此用相同的 key 存储第二个值将覆盖原始值。
    var
        要存储的变量。
    ttl
        存活时间；把 var 存储在缓存中 ttl 秒。在过了 ttl 秒后，被存储的变量将从缓存中删除（在下一次被请求时）。
        如果没有提供 ttl （或者 ttl 是 0），那么该值将一直存在，直到被从缓存中手动删除，或者在缓存中无法存在（清除、重启等）。
    values
        名称在键中，变量在值中。

返回值
    成功时返回 TRUE，失败返回 FALSE。第二种语法会返回带有错误的键的数组。
