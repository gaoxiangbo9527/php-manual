描述
    addslashes ( string $str ) : string

    返回一个字符串，且会在需要被转义的字符前添加反斜线。这些字符是：
        单引号 （'）
        双引号 （"）
        反斜线 （\）
        NUL （空字节）

    addslashes() 的一个用例是去转义字符串中上面提到的字符，这个字符串将被 PHP 计算：
        <?php
        $str = "O'Reilly?";
        eval("echo '" . addslashes($str) . "';");
        ?>

    个人补充：这里有一小段关于 魔术引用 的内容，就不翻译了。

    有时，addslashes() 被错误地用于尝试去防止 SQL 注入。相反，应该使用数据库特定的转义函数和/或预备语句。

参数
    str
        要转义的字符串。

返回值
    返回转义后的字符串。
