描述
    ucwords ( string $str [, string $delimiters = " \t\r\n\f\v" ] ) : string

    返回一个字符串，其中 str 中每个单词的第一个字符大写，如果该字符是一个字母的话。

    单词的定义是紧跟在 delimiters 参数（它们默认是：空格，换页，换行，回车，水平制表和垂直制表）中列出的任何字符后的任何字符串。

参数
    str
        输入字符串。
    delimiters
        可选的 delimiters 包含了单词分隔字符。

返回值
    返回修改后的字符串。

注意事项
    注意：这个函数是区域感知的并且将根据当前所设置的区域处理输入。然而，它只工作在单字节字符集之上。如果你需要使用多字节字符（大多数非西欧语
    言），请查看 multibyte 或 intl 扩展。

    注意：这个函数是二进制安全的。
