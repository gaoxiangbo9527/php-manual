描述
    echo ( string $arg1 [, string $... ] ) : void

    输出所有的参数。不会追加额外的换行符。

    echo 实际上不是一个真正的函数（它是一个语言结构），所以你不需要对它使用括号。echo（不像其它一些语言结构）的行为不像一个函数，所以它不能
    总是用于函数的上下文中。另外，如果想向 echo 传递多个参数，一定不能将参数括在括号内。

    echo 还有一种快捷语法，你可以立即在开始标记后面加上一个等号。在 PHP 5.4.0 之前，这种简短的语法只适用于启用了 short_open_tag 配置设置
    的情况。
        I have <?=$foo?> foo.

    echo 和 print 主要的不同是它接受一个参数列表并且没有一个返回值。

参数
    arg1
        要输出的参数。
    ...

返回值
    没有值被返回值。

注意事项
    注意：因为这是一个语言结构而不是一个函数，所以不能用 变量函数 调用它。

    提示 与在 echo 中使用连接相比，在 echo 中传递多个参数的一个好处是考虑到 PHP 中的句点操作符的优先级。如果传递了多个参数，则不需要圆括号
    来强制执行优先级：
        <?php
        echo "Sum: ", 1 + 2;
        echo "Hello ", isset($name) ? $name : "John Doe", "!";
    使用连接，句点操作符的优先级与加法操作符相同，并且优先级高于三元运算符，所以必须使用括号以得到正确的行为：
        <?php
        echo 'Sum: ' . (1 + 2);
        echo 'Hello ' . (isset($name) ? $name : 'John Doe') . '!';
