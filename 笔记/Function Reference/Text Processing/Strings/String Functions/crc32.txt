描述
    crc32 ( string $str ) : int

    生成 str 的 32 位长度的循环冗余校验和多项式。这通常用于验证正在传输的数据的完整性。

    警告
        因为 PHP 的整型是有符号的，所以在 32 位平台上，许多的 crc32 校验和将会是负整数。然而在 64 位安装上，所有的 crc32() 的结果都是正
        整数。

        所以你需要去使用 sprintf() 或 printf() 的 "%u" 格式符去获取无符号的 crc32() 校验和的十进制字符串表示。

        对于该校验和的十六进制表示，你可以使用  sprintf() 或 printf() 的 "%x" 格式符或者 dechex() 转换函数，这两个函数还负责将
        crc32() 的结果转换为无符号整数。

        我们也考虑过让 64 位安装为更高的结果值返回负整数，但这会破坏十六进制转换，因为负的结果会得到一个额外的 0xffffffff######## 的偏移
        量。因为十六进制表示似乎是最常见的用例，所以我们决定不破坏它，即使它在从 32 位到 64 位时破坏了大约 50% 的直接小数比较。

        回顾一下，让该函数返回一个整数可能不是最好的主意，而立即返回一个十六进制字符串表示（如 md5() 所做的）可能是一个更好的、可开始的计划。

        要想得到一个可移植性更强的解决方案，还可以考虑通用的 hash() 。hash("crc32b", $str) 将会和
        str_pad(dechex(crc32($str)), 8, '0', STR_PAD_LEFT) 返回一样的字符串。

参数
    str
        数据。

返回值
    以一个整数的形式返回 str 的 crc32 校验和。
