描述
    hex2bin ( string $data ) : string

    解码一个十六进制字符串到二进制字符串。

    小心 此函数不会将十六进制数转换为二进制数。可以通过使用 base_convert() 函数来完成这种转换。

参数
    date
        十六进制表示的数据。

返回值
    返回给定数据的二进制表示，或者失败时返回 FALSE .

错误/异常
    如果十六禁止输入字符串是奇数长度或者无效的十六进制字符串，那么会抛出一个 E_WARNING 级别的错误。

变更日志
    版本        描述
    5.5.1	    如果输入字符串是一个无效的十六进制字符串，会抛出一个警告。
    5.4.4	    如果输入字符串的长度为奇数，会抛出一个警告。在 PHP 5.4.0 中，该字符串会被静默的接受，但是最后一个字节会被截断。
