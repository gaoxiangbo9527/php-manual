描述
    strtr ( string $str , string $from , string $to ) : string
    strtr ( string $str , array $replace_pairs ) : string

    如果给定三个参数，这个函数会返回 str 的一个副本，其中 from 中出现的每个（单字节）字符都被转换为 to 中的对应字符，例如，每次出现的
    $from[$n] 都被替换为 $to[$n]，$n 是两个参数中的某个有效的偏移量。

    如果 from 和 to 具有不同的长度，二者中较长的那个参数中的多余字符将被忽略。str 的长度与返回值的长度将会是相同的。

    如果给定两个参数，第二个参数应该是一个形式为 array('from' => 'to', ...) 的数组。返回值是一个字符串，其中所有出现的数组键会被对应的值
    所替换。最长的键将会被先尝试。一旦子字符串被替换，它的新值将不再被搜索。

    在这种情况中，键和值可以是任意的长度，前提是没有空键；另外，返回值的长度可能与 str 的长度不同。然而，当所有的键拥有相同的长度时，这个函数
    将是最有效的。

参数
    str
        要被翻译的字符串。
    from
        将要被翻译为 to 的字符串。
    to
        将替换 from 的字符串。
    replace_pairs
        replace_pairs 可以用于替代 to 和 from，这种情况下，它是一个形式为 array('from' => 'to', ...) 的数组。

返回值
    返回被翻译后的字符串。

    如果 replace_pairs 包含一个空字符串（""）的键，将会返回 FALSE。如果 str 不是一个标量类型，那么它不会被强制转换成一个字符串，而会引发
    一个警告并且返回 NULL。
