描述
    sscanf ( string $str , string $format [, mixed &$... ] ) : mixed

    函数 sscanf() 是 printf() 的输入类比。sscanf() 从 str 字符串读取并根据 format 指定的格式对其进行解释，在 sprintf() 的文档中有
    format 的描述。

    格式字符串中的任何空白都与输入字符串中的任何空白匹配。这意味着即使是格式字符串中的一个制表符 \t 也能匹配上输入字符串中的单个空格字符。

参数
    str
        将要被解析的输入字符串。
    format
        str 的解释格式，在 sprintf() 的文档中有描述，但有以下区别：

        函数是非地区感知的。
        不支持 F，g，G 和 b 。
        D 表示小数。
        i 表示具有进制检测的整型。
        n 表示到目前为止处理的字符数。
        s 停止读取任何空白字符。
    ...
        可选的按引用传入的将包含解析后的值的变量。

返回值
    如果只有两个参数被传递进该函数，解析后的值将作为数组返回。否则，如果传递了可选的参数，该函数将返回被分配值的数量。可选的参数必须通过引用
    传递。

    如果 format 中期望的子字符串比 str 中的可用字符串多，则返回 -1 。
