描述
    stripos ( string $haystack , mixed $needle [, int $offset = 0 ] ) : int

    在 haystack 字符串中查找 needle 第一次出现的数字位置。

    不像 strpos()，stripos() 是大小写不敏感的。

参数
    haystack
        要在其中进行搜索的字符串。
    needle
        注意，needle 可以是有一个或者多个字符的字符串。

        如果 needle 不是一个字符串，它会转换成一个整数并且用作字符的序数值。从 PHP 7.3.0 开始废弃了这种行为，并且依赖于这种行为是非常不推
        荐的。取决于预期的行为，needle 要么应该显式的转换为字符串，要么应该显式的执行对 chr() 的调用。
    offset
        如果指定，搜索将从在字符串的开头统计了此数目的字符后开始。如果偏移量是负数，搜索将从在字符串的末尾统计了此数目的字符后开始。

返回值
    返回相对于 haystack 字符串开头的 needle 存在的位置（不受 offset 的影响）（个人补充：不管从哪个偏移量开始进行搜索，返回的位置始终是相
    对于字符串的开头的）。另请注意，字符串的位置从 0 开始，而不是 1。

    如果没有找到 needle，则返回 FALSE。

    警告 这个函数可能会返回布尔值 FALSE，但也可能返回一个求值为 FALSE 的非布尔值。更多信息请阅读关于 布尔型 的章节。使用 === 以测试这个函
    数的返回值。

更新日志
    版本     描述
    7.3.0	传递一个整型作为 needle 已经被废弃。
    7.1.0   添加了对负数索引的支持。

注意事项
    注意：这个函数是二进制安全的。
