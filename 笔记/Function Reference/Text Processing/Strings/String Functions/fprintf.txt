描述
    fprintf ( resource $handle , string $format [, mixed $... ] ) : int

    写一个根据 format 生成的字符串到由 handle 所指定的流资源。

参数
    handle
        通常是由 fopen() 创建的一个文件系统指针 资源 。
    format
        该格式字符串由零个或多个指令组成：直接复制到结果中的普通字符（不包括 % ）和转换规范，每个转换规范都会获取它自己的参数。

        转换规范遵循下面这个原型：%[argnum$][flags][width][.precision]specifier 。

        Argnum
            后跟一个美元符号 $ 的整数，用于指定在转换中处理哪个编号的参数。
        Flags
            标志	    描述
            -	    在给定的字段宽度内左对齐；默认是右对齐。
            +	    在正数前面加上加号 + ；默认情况下，只有负数以负号 - 作为前缀。
            (space)	用空格填充结果。这是默认值。
            0	    只在左边用 0 填充数字。对于 s 说明符，它也可以右补零。
            '(char)	用字符 (char) 填充结果。
        Width
            一个整数，表示这个转换将产生多少字符（最少）。
        Precision
            一个点号跟着一个整数，该整数的含义取决于说明符：

            对于 e，E，f 和 F 说明符：这是要在小数点后打印的位数（默认为 6）。
            对于 g 和 G 说明符：这是要打印的最大有效位数。
            对于 s 说明符：它作为一个截断点，设置字符串的最大字符限制。

            注意：如果指定了点号，但没有显式的指定 precision 值，则假设为 0 。

            注意：尝试使用一个大于 PHP_INT_MAX 的位置说明符将生成警告。
        Specifiers
            说明符   描述
            %       一个字面的百分比字符。不需要参数。
            b       参数被视为一个整型并且被显示为一个二进制数。
            c	    参数被视为一个整型并且被显示为具有该 ASCII 码的字符。
            d	    参数被视为一个整型并且被显示为一个（有符号的）十进制数。
            e	    参数被视为一个科学计数法（例如 1.2e+2）。从 PHP 5.2.1 开始，precision 说明符表示小数点后的位数。在早期版本中，它
                    被认为是有效数字位数（少了 1 位）。
            E	    像 e 说明符，但使用大写字母（例如 1.2E+2）。
            f	    参数被视为一个浮点型并且被显示为一个浮点数（地区感知）。
            F	    参数被视为一个浮点型并且被显示为一个浮点数（非地区感知）。从 PHP 5.0.3 开始可用。
            g       一般格式。

                    如果 precision 不为 0，让 P 等于 precision，如果 precision 被省略，则 P 为 6，如果 precision 是 0 ，则 P 为
                    1 。然后，如果风格为 E 的转换具有指数 X：

                    如果 P > X ≥ −4，转换使用风格 f 和精度 P − (X + 1) 。否则，转换使用风格 e 和精度 P - 1 。
            G	    像 g 说明符，但使用 E 和 f 。
            o	    参数被视为一个整型并且被显示为一个八进制数。
            s	    参数将被视为一个字符串并且被显示为一个字符串。
            u	    参数被视为一个整型并且被显示为一个无符号的十进制数。
            x	    参数被视为一个整型并且被显示为一个十六进制数（使用小写字母）。
            X	    参数被视为一个整型并且被显示为一个十六进制数（使用大写字母）。

            警告 c 说明符会忽略填充和宽度。

            警告 如果试图将字符串和宽度说明符与每个字符需要多个字节的字符集结合使用，可能会导致意外的结果。

            变量将为说明符提供一个合适的类型：
            类型处理
                类型          说明符
                字符串         s
                整型           d, u, c, o, x, X, b
                浮点型         g, G, e, E, f, F
    ...

返回值
    返回写入的字符串的长度。
