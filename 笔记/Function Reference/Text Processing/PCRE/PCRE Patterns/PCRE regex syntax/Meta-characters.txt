正则表达式的强大来自于能够在模式中包含替换和重复的能力。通过使用 元字符，它们被编码进模式中，元字符并不代表它们自己，而是以某种特殊的方式进行
解释。

有两组不同的元字符：模式内除方括号中的任何地方可以识别的元字符，以及方括号中可以识别的元字符。方括号之外，元字符如下：

方括号之外的元字符
    元字符   描述
    \	    具有多种用途的常规转义字符
    ^	    断言主题（或行，在多行模式下）的开始
    $	    断言主题的结束或在一个终止换行符之前（或行的结束，在多行模式下）
    .	    匹配除换行符以外的任何字符（默认）
    [	    开启字符类定义
    ]	    结束字符类定义
    |	    开启可选择的分支
    (	    开启子模式
    )	    结束子模式
    ?	    扩展 ( 的含义，以及 0 或 1 量词，也使贪婪的量词变得懒惰（见 重复）
    *	    0 或更多 量词
    +	    1 或更多 量词
    {	    开启 最小/最大 量词
    }	    结束 最小/最大 量词

一个模式中位于方括号中的部分被称作 字符类。在字符类中，仅有的元字符是：

方括号（字符类）内的元字符
    元字符   描述
    \	    常规的转义字符
    ^	    取反该（字符）类，但仅当为第一个字符时
    -	    指明字符范围
