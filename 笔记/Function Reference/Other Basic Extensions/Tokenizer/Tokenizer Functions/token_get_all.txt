描述
    token_get_all ( string $code [, int $flags = 0 ] ) : array

    token_get_all() 会使用 Zend 引用的词法扫描器把给定的 code 字符串解析成 PHP 语言令牌。

    有关解析器令牌的列表，参见 解析器令牌列表，或者使用 token_name() 去把一个令牌值翻译成它的字符串表示。

参数
    code
        要解析的 PHP 源码。
    flags
        有效的标志：
            TOKEN_PARSE - 在特定上下文中识别使用保留字的能力。

返回值
    一个令牌标识符的数组。每个令牌标识符个体要么是单个字符（例如，: ;、 .、 >、! 等等），要么是一个包含三个元素的数组，元素 0 零包含着令牌
    的索引，元素 1 中包含了原始令牌的字符串内容并且元素 2 中包含了行号。
