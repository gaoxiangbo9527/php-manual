描述
    iterator_to_array ( Traversable $iterator [, bool $use_keys = TRUE ] ) : array
    将迭代器的元素复制到一个数组中。

参数
    iterator
        将要被复制的迭代器。
    use_keys
        是否使用迭代器元素的键作为索引。

        在 PHP 5.5 或更高版本中，如果键是数组或对象，则会生成一条警告。NULL 键将被转换为空字符串，浮点型键将被截断为对应的整数，资源键将生
        成一条警告并且会被转换成它们的资源 ID ，而布尔型键将会被转换成整型。

        注意：如果没有设置该参数或者设置为 TRUE，重复的键将会被覆盖。具有给定键的最后一个值将在返回的数组中。将这个参数设置为 FALSE 可以在
        任何情况下获得所有的值。

返回值
    一个包含了迭代器元素的数组。

变更日志
    版本        描述
    5.5.0       当启用了 use_keys 参数时，iterator_to_array() 获得了对整数和字符串以外的键类型的支持。
    5.2.1	    增加了 use_keys 参数。
