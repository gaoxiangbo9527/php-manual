描述
    spl_autoload_extensions ([ string $file_extensions ] ) : string
    这个函数能够修改和检查内置的 __autoload() 的回调函数 spl_autoload() 将使用的文件扩展名。
    注意：在被定义的文件扩展名间不应该有空格。
参数
    file_extensions
        当不使用参数调用时，它会简单的返回当前的由逗号分隔的扩展名列表。为了修改文件扩展名列表，简单的使用一个新的、由逗号分隔的文件扩展名列表字符串
        调用该函数即可。
返回值
    一份提供给 spl_autoload() 的由逗号分隔的默认文件扩展名的列表。
    （个人补充，如果传递 $file_extensions 参数，会返回该参数的值，而不是之前的值）
来自用户的笔记
    扩展名不用非得以一个 . 开始，任何形式的扩展名都会被原样追加到名字的后面。
