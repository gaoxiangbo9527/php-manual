描述
    ignore_user_abort ([ bool $value ] ) : int
    设置客户端断开连接是否应引发脚本中止。

    当以命令行脚本的方式运行 PHP 时，如果脚本的 tty 消失在脚本还没有被终止时，那么该脚本会在下次尝试输出任何内容时结束，除非将 value 设置
    为 TRUE 。
参数
    value
        如果设置，该函数会将 ignore_user_abort ini 配置指令设置成 value 给定的值。如果没有设置，该函数将只会返回以前的设置而不会改变它。
返回值
    返回之前的设置，以整数的形式。
注意事项
    在尝试向客户端发送信息之前，PHP 不会去检测用户已经终止连接。
    仅使用 echo 语句不能保证信息被发送，请参阅 flush() 。
