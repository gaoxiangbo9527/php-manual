描述
    exit ([ string $status ] ) : void
    exit ( int $status ) : void

    终止脚本的执行。即使调用了 exit，关闭函数 和 对象的析构函数 也总是会被执行。

    exit 是一个语言结构，如果没有传递任何 status，可以不使用圆括号调用它。

参数
    status
        如果 status 是一个字符串，那么此函数会刚好在退出之前打印此状态。

        如果 status 是一个整数，那个值将会被用作退出状态并且不会被打印。退出状态应该在 0 - 254 范围内，退出状态 255 被 PHP 保留，因此不
        能使用。状态 0 用于成功的终止程序。

返回值
    没有值会被返回。

注意事项
    注意：由于这是一个语言结构而不是一个函数，所以不能用 变量函数 调用它。

    注意：这个语言结构等同于 die() 。
