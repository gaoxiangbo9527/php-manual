描述
    date_parse ( string $date ) : array

    返回关于指定日期的详细信息的关联数组。

参数
    date
        被 strtotime() 接受的格式的日期。

返回值
    成功返回关于被解析日期的信息的数组，或失败返回 FALSE 。

错误/异常
    在日期格式有错误的情况下，（返回的数组中的）元素‘errors’将包含错误消息。

变更日志
    版本    描述
    7.2.0   现在，返回的数组中的元素 zone 表示的是秒而不是分钟，并且它的符号现在是反过来的。例如 -120 现在是 7200 。

相对格式不影响从绝对格式解析而来的值，而是解析到“relative”元素中。
