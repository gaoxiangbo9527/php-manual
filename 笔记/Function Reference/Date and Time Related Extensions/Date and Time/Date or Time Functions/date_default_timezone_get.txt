描述
    date_default_timezone_get ( void ) : string

    按优先次序排列，该函数将按如下顺序返回默认的时区：
        读取使用 date_default_timezone_set() 函数设置的时区（如果有的话）
        PHP 5.4.0 以前：读取 TZ 环境变量（如果不为空）
        读取 date.timezone ini 选项的值（如果有设置）
        PHP 5.4.0 以前：查询操作系统（如果操作系统支持并允许）。这使用了一种猜测时区的算法。这不太可能在任何情况下都能正常工作。当到达此阶
        段时将显示警告。不要依赖它能猜对，应该将 date.timezon 设置为正确的时区。

    如果以上没有一个成功，那么 date_default_timezone_get() 将会返回一个默认的 UTC 时区。

返回值
    返回一个字符串。

变更日志
    版本     描述
    5.4.0   TZ 不在被用于去猜测时区。
    5.4.0	时区不再从操作系统可用的信息中猜测，因为猜测的时区不能依赖。
