PHP 的安全模式试图解决共享服务器的安全问题。尝试在 PHP 级别解决这个问题在体系结构上是不正确的，但是由于在 web 服务器和操作系统级别的替代方案
不太现实，许多人，特别是 ISP，目前使用安全模式。

警告：这个特性在 PHP 5.3.0 被废弃了并且在 PHP 5.4.0 被移除了。

安全模式的变更日志
    版本         描述
    5.4.0	    从 PHP 中移除了，并且在被启用时会生成一个致命的 E_CORE_ERROR 级别的错误。
    5.3.0	    被废弃，并且添加了 E_DEPRECATED 错误。

个人补充：如上所述，就不翻译剩下的内容了。
